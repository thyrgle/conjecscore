Dependencies
============

- [uv](https://docs.astral.sh/uv/)
- [Tailwind-cli](https://tailwindcss.com/docs/installation/tailwind-cli)

Running
=======

To run the server on localhost:8000:

```make
make all
```

To compile frontend:


```make
make frontend
```

To run just the backend:

```make
make backend
```
